# Para Powerlifting 2026 🏋️

競技判定疑似体験アプリ — パラパワーリフティング2026の公式判定シミュレーター

**Live Site:** https://khanhnguyenai.github.io/Para-powerlifting-2026/

---

## 概要

競技前に疑似体験して判定方法を理解し、競技中は観戦しながら判定を予想できるWebアプリです。

## 機能

- 判定シミュレーション（Easy / Normal / Hard）
- シングル・マルチタップ練習モード
- 審判カラー別判定画面（Blue / Green / Orange / Purple / White）
- ルール説明・クレジットページ

## 技術スタック

- HTML / CSS / JavaScript（バニラ）
- Bootstrap 4.5
- Animate.css

## デプロイ

GitHub Actions で自動デプロイ（GitHub Pages）
